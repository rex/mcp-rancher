# Architecture Decision Records

Each ADR captures one significant, hard-to-reverse decision: the context,
the options weighed, the choice, and its consequences. Draft with the
`/adr` command (`.claude/commands/adr.md`), which copies `template.md` to
the next `NNNN-<slug>.md`.

**Status lifecycle:** `proposed` → (PR merges) → `accepted`. Also
`superseded by NNNN` / `deprecated`. A `proposed` ADR is committed and
reviewed like code; its **Decision Outcome is left blank for a human** to
fill before it is accepted.
